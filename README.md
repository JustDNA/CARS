CARS
====

This is a Vehicle Accident Response System's software code.

The Android application (connected to an embedded system which detects the accident) helps to accelerate the reach of medical aid to the victims of accident. This is achieved by sending an alert message to emergency services containing details such as time, accidentâ€™s Geo location and address.
